Hello world program

The hello world program is widely known to be the first program written by new programers. Python has a built in function for displaying text within the console

This function is called: print()

To utilize this function you need to give it what you want to display, this is done by writing in between the brackets. So for the Hello World program it would be: print("Hello World")
